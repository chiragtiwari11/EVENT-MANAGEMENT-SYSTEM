# Event Management System

A web-based event management system built using Flask and SQLite.

## Features
- User login & registration
- Admin dashboard
- Event creation
- Event registration
- View registrations

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- Database: SQLite

## How to Run
1. Install Flask
2. Run `python app.py`
3. Open browser at http://127.0.0.1:5000/
